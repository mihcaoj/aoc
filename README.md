# Advent of Code

My solutions for Advent of Code problems in Go.

## Running solutions

### Clone the repo
```bash
git clone https://github.com/mihcaoj/aoc
```
### Change directory
```bash
cd aoc
```
### Choose year
```bash
cd 2024
```
### Choose day
```bash
cd day02
```
### How to run
```bash
go run .
```
